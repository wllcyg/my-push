## 一、后端（NestJS）

### 🔴 P0 — 场景对话核心接口

- [ ] **1.1** 安装依赖：`openai`（或对应 AI SDK）、`@nestjs/throttler`、`ioredis`
- [ ] **1.2** 创建 `ChatModule`（`chat.module.ts` / `chat.controller.ts` / `chat.service.ts`）
- [ ] **1.3** 实现 `POST /api/chat/stream` — SSE 流式对话接口
  - 接收参数：`{ sceneId, difficulty, messages[], sessionId? }`
  - 返回：SSE 事件流（`text/event-stream`）
  - 认证：复用 `SupabaseAuthGuard`
- [ ] **1.4** 实现 System Prompt 构建逻辑
  - 根据 `sceneId` + `difficulty` 动态拼装 Prompt
  - Prompt 文本先硬编码，后续迁移到数据库
- [ ] **1.5** 实现流式事件协议解析器
  - 将 AI 回复文本解析为结构化事件
  - 事件类型：`text-chunk` / `pinyin-data` / `correction` / `grammar-note` / `turn-done` / `error`
- [ ] **1.6** 接入 AI API（待确认供应商）
  - 支持流式输出（stream: true）
  - 超时配置：120s
  - 重试策略：失败最多重试 1 次

- [ ] **2.3** `GET /api/sessions` — 获取当前用户的会话列表
- [ ] **2.4** `GET /api/sessions/:id/messages` — 获取某会话的消息历史

- [ ] **2.6** 会话归属权校验（防 IDOR）：所有接口校验 `session.userId === currentUser.id`
