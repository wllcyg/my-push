# 图像工具箱 — 产品规划文档

**版本**: v0.1
**平台**: HarmonyOS NEXT
**日期**: 2026-07
**开发顺序**: 第 2 个（证件照工坊上线后）

---

## 定位

**一句话**：日常图片处理全能工具，替代多个分散的单功能 App。

证件照工坊解决的是"证件照"这个垂直场景，图像工具箱解决的是用户**所有日常图片处理需求**，两者互补不重叠。

---

## 功能模块

| 模块 | 核心能力 | 技术方案 |
|---|---|---|
| **图片压缩** | 指定目标文件大小（KB），自动迭代压缩，不让用户手动试 | Image Kit 质量参数二分迭代 |
| **格式转换** | JPG / PNG / WEBP 互转，支持批量 | Image Kit 编解码 |
| **图片裁剪** | 自定义比例裁剪，内置 1:1 / 4:3 / 16:9 等常用比例快选 | Image Kit + 手势交互 |
| **水印添加** | 文字水印/图片水印，自定义位置、大小、透明度，支持批量 | Canvas 叠加绘制 |
| **图片拼接** | 多图垂直/横向拼接、九宫格、自定义间距 | Canvas 多图布局 |
| **去除背景** | 批量抠图，纯色背景替换，人像智能分割 | Core Vision Kit 人像分割 |

## 与证件照工坊的代码复用

**复用率：约 70%**

| 复用模块 | 说明 |
|---|---|
| ImageProcessor 基础封装 | 压缩/裁剪/格式转换逻辑完全复用 |
| Core Vision Kit 人像分割 | 证件照抠图模块直接迁移 |
| FileExporter 文件导出 | DocumentViewPicker 流程完全复用 |
| 订阅/付费体系 | 完全复用，接入同一套 |
| 通用 UI 组件库 | 按钮/弹窗/进度条复用 |

**新增开发内容**：Canvas 水印叠加、多图拼接布局逻辑（约 30%）

---

## 商业模式

- **免费功能**：压缩、格式转换、基础裁剪（每日限量）
- **订阅解锁**：批量处理无限制、水印功能、去背景高级模式
- **定价参考**：月付 6 元 / 年付 30 元（与证件照工坊独立订阅或合并套餐）

## 差异化关键点

> 市面上"图片压缩工具"都是让用户自己拉压缩率滑块，反复试。
> 本产品直接输入目标大小"我要压到 500KB 以内"，自动搞定。
> 这一个细节就能把用户从竞品拉过来。
