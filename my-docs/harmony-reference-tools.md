# 内容速查工具矩阵 — 产品规划文档

**版本**: v0.1
**平台**: HarmonyOS NEXT
**日期**: 2026-07

---

## 为什么做内容速查工具

- **技术极简**：本地 JSON + 搜索/分类 UI，无 API 依赖
- **内容护城河**：数据越全越准，竞品难以快速追上
- **高留存**：用户反复打开查找，不是一次性工具
- **零运维**：数据内置本地，上线后无维护成本

---

## App 1：公式速查手册

**目标用户**：学生、考研考生、教师

### 内容范围

| 分类 | 内容 |
|---|---|
| 数学 | 三角函数、导数、积分、级数、线代 |
| 物理 | 力学/电学/光学/热学 |
| 化学 | 有机反应、化学平衡、溶液计算 |
| 考研数学 | 高数/线代/概率论重点公式汇总 |

### 功能设计
- 分类浏览 + 关键词全文搜索
- 公式 LaTeX 渲染，清晰展示数学符号
- 收藏夹 + 使用说明 + 典型例题（差异化）

### 商业模式
- 免费：数学/物理基础公式
- 订阅：完整公式库 + 例题

## App 2：考公考编速查手册

**目标用户**：公务员/事业单位备考人群（千万级市场）

### 内容范围
- **行测**：数量关系公式、推理规律、言语技巧速记
- **申论**：论点框架、高分句式模板
- **常识**：政治/历史/地理/科技常识
- **时政**：近期重大政策（唯一需要定期更新的内容）

### 功能设计
- 分类速查 + 搜索 + 错题本
- 每日一练：本地题库随机抽取 5 道

### 商业模式
- 免费：行测公式 + 基础常识
- 订阅：申论模板 + 完整题库 + 错题同步

---

## App 3：劳动法律速查手册

**目标用户**：职场人（劳动纠纷场景）、HR

### 内容范围
- 劳动合同（试用期/违约金）
- 工资休假（加班费/年假/病假）
- 离职权益（经济补偿/N+1/非法辞退）
- 仲裁诉讼（流程/时效/举证）

### 核心差异化
- **赔偿计算器**：输入工作年限+月薪，自动算应得补偿
- 场景化检索：不是法条罗列，是"我遇到这个问题该怎么办"

### 商业模式
- 免费：基础法条速查
- 订阅：场景化建议 + 赔偿计算 + 文书模板

## App 4：单位换算全集

**目标用户**：通用，工程/学生/出行人群

| 类别 | 覆盖 |
|---|---|
| 长度/面积/重量 | 中英制双向换算 |
| 温度 | 摄氏/华氏/开尔文 |
| 数据 | B/KB/MB/GB/TB |
| 速度/压力/功率 | 工程常用单位 |

**功能**：输入任意值，所有单位同时显示换算结果

**商业模式**：全部免费（作为引流工具）

## 开发优先级

```
1. 考公考编速查  ← 搜索量最大，付费意愿强
2. 劳动法律速查  ← 痛点强，赔偿计算是差异点
3. 公式速查手册  ← 学生复购率高
4. 单位换算全集  ← 最快出，作为引流
5. 色卡设计工具  ← 垂直精准，设计师付费高
```

## 核心风险

| 风险 | 应对 |
|---|---|
| 内容准确性 | 每条标注来源，提供用户反馈入口 |
| 内容更新 | 法律/政策类做远程数据拉取（唯一联网需求）|
| 竞品模仿 | 例题/场景化建议是护城河，纯表格数据容易抄但深度内容不容易 |
